Nintendo 64 RSP Disasembler
===========================

A standard C program to disasemble Nintendo 64 Reality Signal Processor code.

Dependencies
------------

    sudo apt install cmake

Build
-----

    cmake .
    make

Usage
-----

    rspdis source.bin

This will produce a `code.asm` file.
